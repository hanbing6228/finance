# Finance UI Prototype

Open `index.html` in a browser to view the standalone Personal Finance mobile
app prototype.

The file is a bundled HTML artifact generated from the supplied design file. It
contains the mobile screens, prototype navigation, design system view, and
layout specs in one self-contained page.

Prototype sections:

- Screens
- Prototype
- Design System
- Specs

MVP boundary: this is a static UI prototype only. It does not connect to real
financial accounts, execute payments or transfers, or send data to external
services.
